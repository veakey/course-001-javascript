# INTENTIONS DU PROJET 

> Ce fichier liste les objectifs du projet par ordre chronologique, du plus ancien (#1) au plus récent (#n)

## [#1] Amélioration du jeu du Pendu
- Corriger le fonctionnement du bouton "Proposer" qui ne réagissait pas
- Ajouter un feedback visuel pour la victoire et la défaite
- Permettre à l'utilisateur de choisir le mot à faire deviner avec un input
- Ajouter un bouton "Nouvelle partie" pour réinitialiser le jeu

## [#2] Amélioration du jeu du Robot
- Ajouter une animation avec délai de 100ms entre chaque instruction pour visualiser le mouvement du robot
- Corriger l'orientation du robot : après tournerDroite() depuis 0°, le robot doit être orienté vers le bas (90°), pas vers le haut
